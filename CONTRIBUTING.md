## Development Setup

```bash
git clone https://github.com/ecsplendid/evolution-light-speed.git
cd evolution-light-speed
npm install
npm run dev
```

## Code Style
- Use TypeScript where possible
- Follow existing React patterns
- Add comments for complex evolutionary algorithms
- Keep UI responsive (target 60fps)

## Pull Request Process
1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Test thoroughly (try various L, p, s combinations)
5. Submit PR with clear description

## Scientific Rigor
This is an educational tool, but accuracy matters:
- Cite equations from the paper
- Test edge cases
- Validate against known results
- Don't oversimplify the science
